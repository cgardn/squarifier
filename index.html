Squarifier

Select any number of images, and they'll be made into squares

Multiple images are packaged into a .zip file.

Just tap-and-hold on the downloaded file and you should get an option to "unzip" or "unpack."

Privacy Notice ^

All processing is done on your local device. Your images are not transmitted or saved anywhere. Privacy!

Note: this means that if you refresh the page, you'll lose any links for processed images - so make sure you save them!
